# Báo cáo
## 1. Mô tả thuật toán
- Sử dụng DP Bottom-up để tính giá trị fibonacci thứ n
- Sử dụng 2 biến để rút ngắn bộ nhớ
## 2. Độ phức tạp
- Không gian: O(1) vì chỉ dùng 2 biến để tính toán
- Thời gian: O(N) vì cần lặp từ 2 đến N
